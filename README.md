# ATTDeveloperCompetition2018

Intended to be used with any major browser (i.e [Chrome](https://www.google.com/chrome/?brand=CHBD&gclid=EAIaIQobChMIpJjgpfmD3AIVkNlkCh3s0gZsEAAYASAAEgIb4PD_BwE&gclsrc=aw.ds&dclid=COOrgqf5g9wCFUUDrQYdbOUEBQ), [Edge](https://www.microsoft.com/en-us/windows/microsoft-edge), [Firefox](https://www.mozilla.org/en-US/firefox/new/), etc...) and [Tampermonkey](https://tampermonkey.net/)

Only works on the Google search engine currently

Gets users Google searches and saves them to a MLab database

Returns searches in JSON format when accessing https://attappdevcomp2018.herokuapp.com/result?ipAddress=<your IP Address>
